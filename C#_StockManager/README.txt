StockManager is a C# project which implements a text-based stock management program which allows adding and removing stock items, as well as modifying their prices and quantities.
This is a simple program which deletes its data on exit with no permanent database component.

Requirements.txt - Project specifications and design notes

Flowchart.drawio - Source file for the project flowchart (viewable via diagrams.net)

Flowchart.png - Visual representation of main logical flow of the project's code

StockManager/ - Main project folder containing the C# implementation

Project developed using Visual Studio Code V17.5.2.0
